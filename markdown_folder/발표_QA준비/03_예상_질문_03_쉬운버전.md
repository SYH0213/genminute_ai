## 1️⃣ 챗봇 기본 질문

### Q51. 챗봇이 뭔가요?

**쉬운 답변**:
질문하면 자동으로 답해주는 똑똑한 로봇이에요!

**우리 챗봇의 특징**:
- 회의록을 다 읽고 있어요
- 질문하면 관련 내용 찾아서 답변
- 사람처럼 자연스럽게 대화

**예시 대화**:
```
나: "다음 회의 일정은?"
챗봇: "11월 15일 오후 2시입니다."

나: "누가 발표 담당이야?"
챗봇: "철수가 담당하기로 결정했습니다."

나: "예산은 얼마야?"
챗봇: "총 20만원으로 책정되었습니다."
```

### Q52. 어떻게 답을 아나요?

**쉬운 답변**:
회의록을 미리 다 읽고 기억하고 있다가, 질문하면 관련 부분을 찾아서 답해줘요!

**작동 과정** (4단계):

```
1단계: 질문 받기
  "프로젝트 일정은?"
        ↓
2단계: 관련 내용 검색
  회의록에서 "일정", "스케줄" 관련 내용 찾기
        ↓
3단계: AI가 답변 만들기
  찾은 내용을 바탕으로 자연스러운 문장 작성
        ↓
4단계: 답변 보내기
  "프로젝트 기한은 11월 15일입니다."
```

**비유**:
- 도서관 사서 선생님이 모든 책을 기억하고 있다가
- "이 주제에 관한 책 어디 있어요?" 물으면
- 해당 책을 찾아서 관련 페이지를 보여주는 것!

### Q53. RAG가 뭔가요?

**쉬운 답변**:
"찾아서 답하기" 방식이에요!

**풀어 쓰면**:
- R (Retrieval): 관련 자료 찾기
- A (Augmented): 자료를 활용해서
- G (Generation): 답변 생성하기

**왜 RAG를 쓰나요?**

**일반 AI** ❌:
- 자기가 아는 것만 답변
- 회의록 내용 모름
- 엉뚱한 답변 할 수 있음

**RAG 방식** ✅:
- 회의록을 먼저 찾아봄
- 찾은 내용으로만 답변
- 정확한 답변!

**예시**:
```
질문: "우리 회의에서 결정된 일정은?"

일반 AI: "일반적으로 프로젝트는 2-3주 걸립니다"
         (엉뚱한 답변!)

RAG AI: "11월 15일로 결정되었습니다"
       (회의록에서 찾아서 정확히 답변!)
```

## 2️⃣ 검색 시스템 질문

### Q56. 검색은 어떻게 되나요?

**쉬운 답변**:
단어를 정확히 매칭하는 게 아니라, "의미"로 찾아요!

**일반 검색** (예: Ctrl+F):
```
검색어: "일정"
결과: "일정"이라는 단어가 있는 문장만 찾음
```

**우리 AI 검색**:
```
검색어: "일정"
결과:
- "일정" ✅
- "스케줄" ✅
- "기한" ✅
- "데드라인" ✅
- "마감일" ✅

모두 비슷한 의미라서 다 찾아줌!
```

**비유**:
- 일반 검색: 딱 그 단어만
- AI 검색: 비슷한 뜻도 다 찾아줌

### Q57. 어떻게 의미로 검색하나요?

**쉬운 답변**:
글을 숫자로 바꿔서 비슷한 숫자끼리 찾아요!

**과정**:

**1단계: 글을 숫자로 변환**
```
"프로젝트 일정" → [0.2, 0.8, 0.5, ...]
"마감 기한"    → [0.3, 0.7, 0.4, ...]
```

**2단계: 얼마나 비슷한지 계산**
```
유사도: 95% → 매우 비슷!
```

**3단계: 가장 비슷한 것 선택**
```
상위 3개를 골라서 답변에 사용
```

**비유**:
- 친구들의 "취향 점수"를 매기는 것
- 내 취향과 비슷한 친구 찾기
- 점수가 비슷하면 취향도 비슷!

### Q58. 몇 개를 찾아오나요?

- 실제 회의 대화

- 상위 3개 선택

**2. 요약 내용** (subtopics):
- 주제별 요약
- 전체적인 맥락
- 상위 3개 선택

**왜 6개만?**
- 너무 적으면: 정보 부족
- 너무 많으면: 느려지고 헷갈림
- 6개가 딱 적당! 👍

### 작동 원리 3단계

1. **검색**: 회의록에서 관련 내용 찾기
2. **이해**: AI가 내용 분석
3. **답변**: 자연스러운 문장으로 설명

### 잘 사용하는 법

1. 명확한 질문하기
2. 한 번에 하나씩
3. 못 찾으면 다른 단어로 다시 질문

## 💬 실제 사용 예시

### 시나리오: 회의 후 정보 찾기

**상황**: 1주일 전 회의 내용이 기억 안 남

```
나: "지난 주 결정 사항이 뭐였지?"
챗봇: "프로젝트 기한 15일로 확정,
      철수가 발표 담당으로 결정되었습니다."

나: "예산은?"
챗봇: "총 20만원으로 책정되었습니다."

나: "내가 뭐 하기로 했더라?"
챗봇: "영희님은 기획서 검토를
      담당하기로 했습니다."
```

**결과**: 5분 안에 필요한 정보 모두 확인! ✅
